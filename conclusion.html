¿Para que utilizamos los servidores?


En esta unidad tuvimos que configurar un servidor con los servicios de: DOMINIO, DHCP, IIS, MYSQL E IMPRESION, todo esto con la finalidad de emular un entorno real de una empresa, el dominio nos permite poder reconocer de manera mas sencilla el servidor, pudiendo conectarnos a el mediante un nombre de dominio y no su direccion IP, la cual configuramos de manera estatica para evitar futuros problemas.

De igual manera configuramos el servicio DHCP para poder asignar de manera automatizada una direccion IP a cada terminal que se conecte a nuestra red, el servicio de IIS nos permite alojar paginas web, tambien utilizamos una pequeña base de datos con MYSQL, el cual configuramos en el servidor y desde las terminales pudimos crear, modificar y eliminar la informacion almacenada. por ultimo utilizamos el servicio de impresion que nos permite configurar una impresora en el servidor para poder compartirla con todas las terminales conectadas.

Consideramos que es de vital importancia adquirir todo este conocimiento, ya que todas las empresas requieren de esta tecnologia.
